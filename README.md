# Student Task Manager

A full-stack web application to manage daily tasks with priorities and due dates. Built as a Capstone Project to demonstrate end-to-end development skills.

## Live Demo
- **Frontend:** [Link to your Vercel URL]
- **Backend:** [Link to your Render URL]

## Tech Stack
- **Frontend:** React (Vite), CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Deployment:** Vercel (Client) & Render (Server)

## Features
- **Create Tasks:** Add title, description, priority, and due date.
- **Manage Status:** Toggle tasks as completed/pending.
- **Filter:** View all tasks or filter by completion status.
- **Delete:** Remove unwanted tasks.
- **Responsive Design:** Works on mobile and desktop.
